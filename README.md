# Break Enforcer

Windows-приложение, которое блокирует экран каждые 60 минут на 3 минуты для разминки. Помогает не забывать вставать из-за компьютера.

## Возможности

- Полноэкранный оверлей с обратным отсчётом
- Блокировка клавиатуры (Alt+Tab, Win, Alt+F4) во время перерыва
- 73 упражнения — 4 случайных на каждый перерыв
- Иконка в трее с оставшимся временем
- Отложение перерыва (до 2 раз по 10 мин)
- Детекция активности — таймер на паузе если вы отошли
- Визуальное предупреждение за 30 секунд — экран сереет с обратным отсчётом, можно продолжать работать
- Обеденный перерыв — полная блокировка экрана по расписанию (настраиваемое время начала и конца)
- Настройки через UI (все параметры, список упражнений, обеденный перерыв)
- Автозапуск при старте Windows

## Установка

### Вариант 1: Готовый .exe (рекомендуется)

1. Скачайте папку `dist/BreakEnforcer/` или соберите сами (см. ниже)
2. Запустите `BreakEnforcer.exe`
3. Приложение появится в системном трее

### Вариант 2: Из исходников

```bash
# Требуется Python 3.10+
pip install -r requirements.txt
python main.py
```

### Сборка .exe

```bash
# Требуется Python 3.10+
build.bat
```

Результат: `dist/BreakEnforcer/` — папку можно перенести на любой ПК с Windows.

## Настройки

Правый клик по иконке в трее → **Настройки**

| Параметр | По умолчанию | Описание |
|----------|:------------:|----------|
| Время работы | 60 мин | Интервал между перерывами |
| Длительность перерыва | 3 мин | Сколько длится обычный перерыв |
| Перерыв после отложения | 5 мин | Перерыв если было отложение |
| Время отсрочки | 10 мин | На сколько откладывается перерыв |
| Макс. отложений | 2 | Сколько раз можно отложить |
| Порог неактивности | 5 мин | Через сколько бездействия таймер на паузе |
| Упражнения | 73 шт | Список упражнений для перерывов |
| Обеденный перерыв | Выкл | Включить/выключить блокировку на обед |
| Начало обеда | 13:00 | Время начала обеденного перерыва |
| Конец обеда | 14:00 | Время окончания обеденного перерыва |

Настройки хранятся в `%APPDATA%\BreakEnforcer\settings.json`.

## Автозапуск

При каждом запуске приложение прописывается в реестр Windows:
```
HKCU\Software\Microsoft\Windows\CurrentVersion\Run\BreakEnforcer
```

Чтобы убрать из автозагрузки — удалите этот ключ или используйте **Диспетчер задач → Автозагрузка**.

## Структура проекта

```
main.py              — точка входа, оркестрация
settings.py          — менеджер настроек (JSON в AppData)
settings_window.py   — UI окна настроек
config.py            — визуальные константы (цвета, шрифты)
activity.py          — детекция активности пользователя
keyboard_hook.py     — блокировка системных клавиш
overlay.py           — полноэкранный оверлей перерыва
warning_overlay.py   — предупреждение за 30 сек (click-through)
tray.py              — иконка в системном трее
build.bat            — скрипт сборки .exe
requirements.txt     — зависимости Python
```
